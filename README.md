

# AI Attendance Management System

## Project Overview

AI Attendance Management System is a simple Python-based application developed to manage student attendance efficiently. The system allows users to mark students as Present or Absent, save attendance records to an Excel file, and generate attendance reports through a menu-driven interface.

## Features

* Mark attendance as Present or Absent
* Store attendance records with date and time
* Save attendance data in Excel format (.xlsx)
* Display present students
* Display absent students
* Show total number of students
* Calculate attendance percentage
* User-friendly menu-driven system

## Technologies Used

* Python
* Pandas
* OpenPyXL
* Jupyter Notebook

## Requirements

Install the required libraries:

```bash
pip install pandas openpyxl
```

## How to Run

1. Open Jupyter Notebook or any Python IDE.
2. Install the required libraries.
3. Run the Python script.
4. Enter attendance status (P for Present, A for Absent).
5. View attendance reports from the menu options.
